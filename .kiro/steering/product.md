# Product Overview

学習計画カンバン（LPK）は、教科×ステータスの2次元ボードで学習タスクを運用するPWA。1週間スプリントを前提に、日次/週次の計画・実行・振り返りを回す。

## Core Capabilities
- 固定ステータス（Backlog/Today/InPro/OnHold/Done/WontFix）によるカンバン運用とドラッグ操作
- スプリント/カレンダー連動（バーンダウン、週次サマリ、Today負荷の可視化）
- 時間管理（予定/残り時間ゲージ、InProの円形進捗、PT/実績記録）
- オフライン対応＋Google Drive同期、バージョン自動アップデート
- 観覧専用ビューと通知（期日リマインド）、Google Calendar連携

## Target Use Cases
- 対象: 小学4年生〜高校生
- 週次スプリントで学習計画を立て、日々の負荷と期限を自分で調整する習慣をつけたい児童・生徒
- 保護者向けに閲覧権限のみで学習の進捗を共有したい家庭
- マルチデバイス（PC/タブレット/スマホ）で途切れず計画を確認・更新したい利用者

## Value Proposition
- 専用サーバ不要（GitHub Pages配信＋Google Driveストレージ）の軽量構成
- 予定/残り時間のゲージ表示とInPro専用の可視化で「今やるべき」への集中を支援
- スプリント計画〜日次実行〜週次振り返りを一貫したUI/ステータスルールで運用可能
